Get Started

Easy to Use

Unlike the original release, PGM 1.8 is much easier to use. It has no backend, website or API to make hosting more cost-effective.

Open Source

Not only is PGM easy to use, but contributing requires no dependency installation. The plugin is designed with reliability, stability and performance in mind, all made possible thanks to our wonderful contributors.

Active Community

Be part of a community of over 20 contributors and help keep the project alive! Our Discord server is the best place to discuss bug fixes and implementations in real time.
