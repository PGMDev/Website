The plugin's commands are divided in two sections: User commands and Admin commands. Keep in mind that in order to run Admin commands, you must have the `events.staff` permission.

### User commands

<div className="table-container">
  <table>
    <thead>
      <tr>
        <th>Command</th>
        <th>Parameters</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>
          <label>/veto</label>
        </td>
        <td>{`<number>`}</td>
        <td>
          Vetoes a map if a veto round is running. It can only be executed by
          players on a team.
        </td>
      </tr>
      <tr>
        <td>
          <label>/score</label>
        </td>
        <td />
        <td>Shows the current score in the event.</td>
      </tr>
      <tr>
        <td>
          <label>/rounds</label>
        </td>
        <td />
        <td>Shows past and future to-be-played rounds.</td>
      </tr>
    </tbody>
  </table>
</div>

### Admin Commands

        </td>
        <td />
        <td>Lists all loaded teams.</td>
      </tr>
      <tr>
        <td>
          <label>/tm info</label>
        </td>
        <td>{`<team>`}</td>
        <td>Shows information about a team.</td>
      </tr>
      <tr>
        <td>
          <label>/tm unregisterall</label>
        </td>
        <td />
        <td>Clears all registered teams.</td>
