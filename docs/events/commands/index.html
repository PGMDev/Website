Commands

The plugin's commands are divided in 2 sections: User commands and Admin commands. Keep in mind that in order to run Admin commands you must have the events.staff permission.

#User commands

CommandParametersDescription
<number>Vetoes a map if a veto round is running. It can only be executed by players on a team.
Shows the current score in the event.
Shows past and future to-be-played rounds.

#Admin Commands

CommandParametersDescription
<format>Creates a match based on a format.
<team>Registers a team for a match.
Lists all loaded teams.
<team>Shows information about a team.
Clears all registered teams.
