Time Periods

Time periods are numbers suffixed with a time unit. Periods with no formatter suffix default to seconds.

In some contexts, oo can be used to indicate an infinite time period.

SuffixDuration
Day
Hour
Minute
Seconds

Examples

<!-- 5 minutes -->
<time>5m</time>
