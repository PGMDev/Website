Material Matchers​

Depending on the material/item element used a material or multiple materials can be specified with or without data values.

TypeAccepted ValuesExample
Material MatcherA material name and data value separated by a colon (:). If no data value is specified, it will match all variations of the material.wool:0 (Only white wool)
wool (All wool types)
Multi Material MatcherA list of Material Matchers separated by a semicolon (;).wool;log:0 (All wool and oak logs)
