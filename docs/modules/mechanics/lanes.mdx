---

Lanes are used in Race for Wool (RFW) style maps with two parallel lanes for the teams. 
They prevent players from leaving their team's lane and entering the enemies' lane.

#### Lanes Element

<div className="table-container">
  | Element | Description |
  |---|---|
  | `<lanes> </lanes>` | Node containing this world's lanes. |
  
  | Sub-elements || Value/Children |
  |---|---|---|
  | `<lane> </lane>` | A single lane definition. | [Regions](/docs/modules/mechanics/regions) |
</div>

#### Lane Attributes

<div className="table-container">
  | Attribute | Description | Value |
  |---|---|---|
  | `team` | <span className="badge badge--danger">Required</span>The team the lane belongs to and in which they have to stay inside. | [Team ID](/docs/modules/format/teams) |
</div>

### Examples

```xml
<lanes>
    <lane team="blue">
        <region id="blue-lane-total"/>
        <region id="blue-spawn-rooms"/>
    </lane>
    <lane team="red">
        <region id="red-lane-total"/>
        <region id="red-spawn-rooms"/>
    </lane>
</lanes>
```
