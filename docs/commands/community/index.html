Community Commands

This page describes a list of commands, aliases and permissions for Community, a standalone plugin for managing PGM servers. Commands for moderation, punishments, etc will be shown here. Permissions are prefixed with CommunityPermissions (ie CommunityPermissions.LOOKUP_OTHERS).
